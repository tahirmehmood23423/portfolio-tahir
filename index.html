An end-to-end AI-powered job application assistant — designed, built, and deployed solo in 3 weeks. The product automates the five most time-consuming parts of job hunting:

Module 1 — Résumé Parser: Uploads a PDF/DOCX résumé and returns fully structured JSON — personal info, skills, experience, education, projects, certifications.
Module 2 — Job Matcher: Two-pass scoring using Gemini embeddings (semantic similarity) + LLM requirement extraction. Returns a 0–100 match score with per-requirement breakdown.
Module 3 — Résumé Tailorer: Source-bound LLM rewriting — reorders, emphasises, and rewrites existing content for a specific job. Never invents information. Includes ATS compatibility checker.
Module 4 — Cover Letter Generator: Two-pass pipeline: extract strongest talking points → write 4-paragraph letter in chosen tone (Professional / Conversational / Confident / Enthusiastic).
Module 5 — Job Discovery: Scrapes LinkedIn and Remotive in parallel, auto-extracts keywords from résumé, scores and ranks jobs. Zero LLM cost — pure keyword matching.
